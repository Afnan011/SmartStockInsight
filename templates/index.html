About the Project

SmartStock Insight is a web application for predicting stock prices using sentiment analysis and historical stock data. Our advanced machine learning models combine LSTM, GRU, and CNN layers to provide accurate predictions and recommendations.

Key Features

  • Stock selection and detailed information
  • Sentiment analysis from news articles
  • Historical data analysis
  • Price predictions and recommendations
  • Interactive live stock price charts
Sentiment Analysis

Integrate sentiment analysis results from news articles to enhance prediction accuracy.

Machine Learning Models

Utilize advanced LSTM, GRU, and CNN models for precise stock price predictions.

Live Charts

View interactive charts with options for line or candlestick representations of stock data.
